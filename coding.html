    What is HTML and why is it important

    HTML stand for hypertext markup language. It is use for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages. A man name Tim Berners-Lee a contracter at CERN made a prototype ENQUIRE a system for CERN to use and share documents. He wrote a memo propsing an Internet-based hypertext system. The language is written in the form of HTML elements consisting of tags enclosed in angle brackets .

    What is CSS and why is it import

    CSS stands for Cascading Style Sheet and is used by web pages to help keep information in the proper display format. CSS was created to help people have an easier time for their websites. CSS is a style sheet language used for describing the look and formatting of a document written in a markup language.

    How do tags work

    Tags are the hidden keywords within a web page that define how the browser must format and display the content. A tag is written by using greater than or less than symbols and placing a tag between the symbo but the tag must have meanig in HTML.

    What are attributes and how do they work

    Attributes provide additional information about an element and are always specified in the start tag

    You can change an attribute by changing the style of the wirds or background.

    Description of 4 tags

    Head is a tag which defines information about the document. Its function is to include a title for the document, scripts, styles, meta information, and more.

    Em is a tag which defines emphasized text. Its function is often used to indicate an implicit or explicit contrast

    Nav is tag which defines navigation links. This is one of the most import things to have in a website without this you would not go to the other pages.

    P is tag which defines a paragraph This helps you serperate your paragraph on web page.
